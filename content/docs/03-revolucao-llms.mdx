# A revolução da IA: os modelos de linguagem (LLMs)

Como vimos nas aulas anteriores, os LLMs ou Modelos de Linguagem de Grande Escala, estão entre os principais responsáveis pelo boom da IA atual.

Nessa aula, vamos explorar como esses modelos funcionam e como eles estão transformando a interação com a tecnologia.

Em primeiro lugar, precisamos aceitar que esse é um tópico complexo. Os LLMs são frutos de décadas de pesquisa e investimentos bilionários. Não é possível se tornar um especialista nesse tópico com apenas um curso.

Mas a parte boa é que não é preciso ser um especialista para poder se benecificar dessa tecnologia. Podemos fazer um paralelo com carros e engenharia mecânica. A grande maioria das pessoas não sabe como funciona um motor de combustão, mas sabe dirigir e se beneficia desse conhecimento. Ao mesmo tempo, para dirigir é importante entender o básico, por exemplo você deve saber que você não pode colocar diesel em um motor de gasolina.

O objetivo aqui é fazer o mesmo com os LLMs. Vamos entender o básico necessário para conseguirmos usar essa tecnologia de forma consciente e segura.

## O que são LLMs?

LLMs são modelos de linguagem, ou seja, são programas de computador que foram desenvolvidos para entender e gerar texto de uma forma que faça sentido para nós humanos.

A diferença entre um LLM e outro programa qualquer de computador, é que o LLM utiliza como base algoritmos especiais de aprendizado de máquina, chamados de redes neurais profundas.

Essas redes neurais precisam de uma enorme quantidade de dados para "aprender" a gerar texto de forma coerente.

Vamos fazer uma analogia para entender melhor como esses modelos funcionam.

Imagine um cenário onde não existem computadores ou internet. Todo o conhecimento do mundo está disponível em livros dentro de uma biblioteca. Como são muitos livros, é muito difícil achar o que você está procurando.

Nesse cenário, alguém teve a ideia de selecionar um "oráculo". Esse oráculo é uma pessoa muito inteligente que vai ler todos os livros e fornecer respostas para as perguntas que forem feitas a ela.

A questão é que, mesmo sendo muito inteligente, o oráculo não vai conseguir decorar todos os livros. Então ele vai precisar de algum método para conseguir responder as perguntas.

A ideia que ele tem é fazer resumos de cada livro para facilitar a busca por informações. Então se alguém perguntar a ele "Por que o céu é azul?" o oráculo vai lembrar dos livros que leu e consultar seus resumos para fornecer uma resposta.

Os LLMs funcionam mais ou menos da mesma forma. Eles têm acesso a uma biblioteca enorme de dados em formato digital que vêm da internet e de outras fontes. Então eles passam por um processo chamado de "treinamento" onde eles "lêem" todos os dados e fazem correlações entre eles.

Depois do treinamento, o LLM é capaz de responder perguntas sobre o que ele aprendeu. A diferença está na forma como ele responde as perguntas.

Enquanto o oráculo consultaria os seus resumos, o LLM funciona de forma "matemática". Ele não é capaz de pensar como um ser humano. Ele não entende o que é "céu azul", ele nunca saiu na rua e olhou para cima para ver o céu. 

O que o LLM faz é utilizar cálculos estatísticos com base no seu treinamento para gerar a próxima palavra de um texto. Quando ele está respondendo a pergunta "que cor é o céu?" ele vai começar a gerar cada palavra "o céu é ..." e de acordo com os dados que ele aprendeu, ele consegue "entender" que a palavra "azul" se correlaciona muito com a palavra "céu" nesse contexto. Então ele vai saber responder que o céu é azul.

Parece loucura, né? Mas isso só é possível por conta da grande quantidade de dados que ele foi treinado. E quanto mais dados ele receber, melhor ele vai ser. É por isso que hoje em dia, dados são tão valiosos.

## Por que os LLMs são tão revolucionários?

Agora você já entendeu o que é um LLM e como ele funciona. Mas por que ele é tão revolucionário?

A resposta é simples: ele é capaz de compreender e gerar texto de forma muito natural e fluida.

Quando você "conversa" com ele, parece que você está falando com um humano. Lembra do Teste de Turing? Pois é, os LLMs passam nesse teste tranquilamente.

A partir da capacidade de gerar texto, os LLMs podem ser utilizados para diversas finalidades.

Você pode fazer perguntas sobre qualquer assunto. Você pode conversar com ele como se fosse um amigo. Você pode pedir ajuda para escrever um texto, um email, um relatório, e ele vai fazer isso muito bem.

Além disso, essa mesma técnica de gerar texto pode ser utilizada para gerar imagens e áudios. Ou seja, você pode pedir para ele gerar uma imagem de um cachorro, e ele vai fazer isso.

É daí que vem o termo "IA Generativa".

A grande limitação desses modelos é que eles são muito "fechados" dentro de seu treinamento. Eles não tem acesso a informações externas aos dados que foram usados para treiná-los.

E é nesse ponto que essa tecnologia está evoluindo agora. Já existem técnicas para "conectar" esses modelos com a internet e outras fontes de dados para torná-los mais inteligentes e poderosos.

Ao longo do curso, nós vamos ver alguns exemplos disso na prática.
