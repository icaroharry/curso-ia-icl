# IA para pesquisa e aprendizado

Uma das aplicações mais interessantes da IA é para o aprendizado e pesquisa.

Ao conversar com um modelo de linguagem, você tem acesso a milhares de fontes de informação condensadas em um único lugar. Isso pode ser muito útil para você incrementar o seu estudo.

Mas antes de aprendermos algumas técnicas, é importante entender que é preciso ter muito cuidado com a forma como vamos interagir com o modelo. Como vimos nas aulas anteriores, os LLMs não podem ser levados como verdade absoluta, afinal eles estão sujeitos a erros.

Sempre que você usar um LLM para estudar, lembre-se de validar as informações que ele te fornece.

Dito isso, vamos começar com um exemplo prático.

## Exemplo 1: Estudando para provas e concursos

Vamos supor que você está estudando para a prova da FUVEST (https://www.fuvest.br/wp-content/uploads/fuvest2024_primeira_fase_prova_V.pdf).

Para isso, você pode usar o ChatGPT para gerar um plano de estudos.

```
Estou estudando para a prova da FUVEST. Gere um plano de estudos completo para me ajudar a estudar.
```

Outra coisa que você pode fazer, é anexar documentos ao seu prompt. Os modelos de linguagem possuem um limite da quantidade informações que você pode passar para ele, mas no geral, esse limite é suficiente para anexar documentos que vão tornar a sua resposta mais interessante.

## Exemplo 2: Perguntas e respostas

Você pode pedir para que o modelo seja uma espécie de tutor para fazer perguntas e avaliar suas respostas.

```
Estou estudando para uma prova de geografia. Quero que você me faça perguntas e avalie a minha resposta depois de cada pergunta.
```

## Exemplo 3: Pesquisa na web

Vimos um exemplo parecido na aula anterior, mas você pode usar os modelos mais modernos para te ajudarem a fazer uma pesquisa na web.

Por exemplo, vamos supor que você seja um investidor da bolsa de valores e queira saber quais são as últimas notícias que afetaram o preço de alguma ação.

```
Quais são as últimas notícias que afetaram o preço da ação da Apple?
```

## Conclusão

Com esses exemplos, você pode perceber que a IA é capaz de te ajudar a incrementar os seus estudos e gerar aprendizado para você.

Um dos segredos para conseguir fazer isso de maneira eficiente, é escrever bons prompts. A forma com você faz perguntas ao modelo está diretamente relacionada com o resultado que você vai conseguir. Por isso, uma dica interessante é experimentar diferentes formas de fazer perguntas ao modelo e anotar quais são as mais efetivas.
